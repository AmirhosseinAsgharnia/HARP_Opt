HARP_Opt: An optimization tool for wind and water turbines.

Current Maintainers

  • Danny Sale (University of Washington)
  • Michael Lawson (National Renewable Energy Laboratory)

User Information

If you came to this page directly without going through the NWTC Information Portal, we would appreciate if you could report your user information before cloning the repository. We use this information in order to allocate resources for supporting our software, and to notify users of critical updates.

Prerequisites

Users:

  • Matlab Component Runtime v7.09

Developers:

  • C compiler
  • Fortran compiler
  • WT_Perf
  • NWTC_Library
  • Matlab (optional toolboxes: Optimization, Global Optimization, Parallel Computing)

Run Examples

From the Matlab prompt (compatible with Linux, Windows, and Mac), 

$ HARP_Opt input_filename.inp

or from the Windows command prompt (using the compiled version)

$ HARP_Opt_windows.exe input_filename.inp
